  • 08:00 - 09:00

    Credenciamento e coffee-break

  • 09:00 - 10:00

    Escalando sua aplicação para milhões de visitas

    Luiz Fernando (Zizaco)

    Escalabilidade Como utilizar cache de forma inteligente para otimizar o uso de recursos e acelerar a sua aplicação sem comprometer a experiência do usuário. (Case da Leroy Merlin)

    Vuelando nas CryptoCoins

    Walker de Alencar

    Exemplo prático do uso de Laravel+Vue+Materialize em projeto com Wallets da Decred (Moeda digital criada a partir de um racha do core do Bitcoin).

  • 11:30 - 12:00

    Superpoderes com Collections

    Mateus Guimarães

    Minipalestra explicando o benefício de usar Collections (Tanto Eloquent Collections quanto as Collections de suporte) e como é possível deixar o código menor e mais legível usando-as

  • 13:30 - 14:30

    Construindo e mantendo aplicações multi-tenant

    Aryel Tupinambá

    O multi-tenancy é uma evolução natural para boa parte das aplicações. Startups e seus produtos precisam dele para seu modelo SaaS, empresas e seus sistemas o querem para o reuso de código. Essa palestra irá mostrar técnicas e modelos para a aplicação de multi-tenancy, e erros comuns no processo. Será exemplificado com um estudo de caso de um projeto nosso, mostrando como lidamos com uma dívida técnica imensa e arquitetura limitada, até chegar em uma plataforma sólida, automatizada e customizável

  • 14:30 - 15:30

    Os Três Mosqueteiros: Vagrant - Git - Deploy

    Edison Pereira da Costa

  • 15:30 - 16:00

    Coffee-break

  • 16:00 - 17:00

    Laravel e front end: dicas para dominar a integração com Gulp/Elixir

    Fábio Vedovelli

  • 17:00 - 18:00

    Laravel 5.3 e Aplicações Real Time

    Diego Hernandes

    Dentre as novidades do Laravel 5.3, está o Laravel Echo, um poderoso componente de broadcast de eventos desenvolvido para facilitar o desenvolvimento de aplicações real-time

Fábio Vedovelli

Fábio Vedovelli é desenvolvedor desde 2000 e programa em PHP e Javascript. Entusiasta de Laravel Framework desde o primeiro contato em 2013, já desenvolveu diversos projetos no Brasil e exterior usando a ferramenta. É o criador e responsável pelo site Vedcasts.com.br, no qual dispobiliza video aulas sobre programação para a web.

Luiz Fernando (Zizaco)

Luiz Fernando (Zizaco) já trabalhou com diversas linguagens e frameworks populares como Codeigniter, CakePHP, Zend e Rails. Autor de diversos artigos, tutoriais e projetos opensource, além de ser palestrante nos hangouts da comunidade Laravel Brasil e de ter feito apresentações em comunidades como MongoDB User Group, Caravana web, Ruby User Group (GURU Sorocaba), entre outros.

Desenvolvedor pioneiro de Laravel no Brasil, trabalha com o framework no dia-a-dia, com vivência e pragmatismo em desenvolvimento web utilizando diversas tecnologias.

Isaque Souza

Isaque Souza é desenvolvedor de sistemas com PHP e Javascript desde 2008, sempre focado em inovações tecnológicas e com passagens por diversos frameworks de mercado como Zend, Laravel, CodeIgniter e outros. Consultor em Zend e Laravel, já participou de projetos públicos e privados para FGV, Zurich, IdealInvest e recentemente responsável pela versão 1.0 do Laravel PagSeguro.
